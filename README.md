# joho-judge-system

学生が作成したJupyterNotebook内の課題を自動評価するためのプログラム



## 要件
- npm install axios vue-axios
- pip install flask flask-cors nbformat nbconvert
- pip install jinja2==3.0.3
- 内部サーバーはhttp://localhost:5000で融通


## 諸々の操作

### Projectの初回起動

```sh
npm install
```

### 内部サーバーの起動

```sh
npm run dev
```

### アプリケーションのビルド

```sh
npm run build
```
### Python経由での起動
```
python main.py
```
